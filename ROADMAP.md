# Roadmap

## v0.2 — Integration-ready intelligence (current)

- Connection and asset-mapping contracts
- Multi-channel normalization and data-quality gates
- AI channel evaluation, anomalies, attribution, scoring, and approval
- Mock connectors and deterministic utilities

## v0.3 — Community provider adapters

- Read-only Search Console and GA4 reference adapters
- WordPress draft-only reference connector
- Connector contract tests and sanitized provider fixtures
- Metric-definition and schema migration registry

## v0.4 — Dashboard interoperability

- API contract for projects, connections, evaluations, experiments, and approvals
- Export formats for common dashboards and task systems
- Scheduled weekly and monthly review payloads

## v0.5 — Evidence-based automation

- Narrow, reversible execution adapters
- Approval signatures, expiry, verification, and rollback tests
- Experiment registry and cross-project learning library

No milestone will add unrestricted autonomous publishing, spending, messaging, deletion, or access management.

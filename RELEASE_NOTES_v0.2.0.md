# Duan Growth Skills v0.2.0

## Multi-channel AI growth intelligence

This release expands Duan Growth Skills from a growth reasoning toolkit into an integration-ready operating layer for marketing and growth systems.

### Highlights

- 30 portable skills and 9 optional Claude Code agents
- Secure account-connection and project-asset mapping workflows
- Connection health and data-quality gates before AI conclusions
- Unified but lineage-preserving multi-channel data contracts
- Detailed SEO, paid media, social, local search, and CRM analysis
- Cross-channel attribution reconciliation without false precision
- Explainable AI evaluation cards, anomaly alerts, and health scores
- Approval-safe action planning with preview, limits, verification, and rollback
- Sanitized mock connector, examples, and five public benchmark cases

### Important boundary

This repository does not ship real OAuth credentials or unrestricted production adapters. It supplies skills, contracts, fixtures, and deterministic utilities. A deployed application must implement provider authorization, encrypted token storage, collectors, scheduling, user roles, project isolation, and approved execution.

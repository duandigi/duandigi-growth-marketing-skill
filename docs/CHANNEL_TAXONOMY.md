# Channel taxonomy

Canonical channels are a reporting contract, not a replacement for provider dimensions.

Recommended top-level values:

- organic_search
- paid_search
- paid_social
- organic_social
- local_search
- email
- crm_lifecycle
- referral
- affiliate_partner
- community
- direct_unknown
- display_video
- marketplace
- offline
- product_owned

Store the original provider channel, source, medium, campaign, content, and targeting fields alongside the canonical value. Project-specific subchannels are allowed when documented.

# Case 01 — Connection versus performance

## Prompt

Evaluate this project for the last seven days. GA4 synced today. Search Console last synced five days ago. Meta Ads returns a permissions error. Website sessions are stable, Search Console clicks appear down 42%, and Meta leads appear as zero. What happened, and what should we do?

## Expected strengths

- Does not claim SEO and Meta performance collapsed.
- Classifies stale and unavailable connections before performance.
- Identifies which analyses remain possible from GA4.
- Provides provider-specific remediation and an approval-safe next step.

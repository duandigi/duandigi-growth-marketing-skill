# Changelog

## [0.2.0] - 2026-07-11

### Added

- Eighteen skills for account integration, asset mapping, permissions, connection health, data quality, normalization, channel analysis, anomaly detection, attribution, AI evaluation, health scoring, executive reporting, and action approval
- Five optional Claude Code agents for integration, data quality, channel analysis, cross-channel evaluation, and approval control
- Provider capability registry for Search Console, GA4, Google Ads, Meta, WordPress, CRM, and signed webhooks
- Mock connector and sanitized multi-channel fixtures
- JSON contracts for connections, mappings, normalized records, evaluation cards, health scores, anomalies, and approvals
- Deterministic utilities for connection validation, normalization, anomaly detection, and health scoring
- Architecture, OAuth, data model, AI evaluation, channel taxonomy, approval, and implementation documentation

### Changed

- Expanded the project from growth reasoning to integration-ready multi-channel growth intelligence
- Increased the validated skill catalog from 12 to 30
- Strengthened security, data-isolation, approval, and confidence requirements

### Security

- Explicitly forbids credentials in skills, prompts, examples, logs, and commits
- Defaults provider connections to observe mode
- Separates recommendation, preparation, human approval, execution, verification, and rollback
